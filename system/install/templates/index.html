What kind of databases are supported

At this time only MySQL databases are supported. There are no plans to support others.

What kind of table should I create?

Its recommended and tested to use an InnoDB (standard with MySQL) with the charset "utf8" and collation "utf8_bin". You may run into errors using something different.
